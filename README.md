# 🌐 Mi Portafolio Web — Mario Buendia

Este repositorio contiene el código fuente de mi sitio web profesional y portafolio, desplegado en GitHub Pages. El objetivo de este espacio es centralizar y dar visibilidad a mis proyectos terminados en desarrollo de software, automatización y análisis de datos.

🔗 **Enlace en vivo:** [buendiamariodev.github.io](https://buendiamariodev.github.io/)

---

## 🛠️ Proyectos Realizados

### ⏱️ Proyecto 1 — To-Do App
Aplicación web funcional para la gestión y seguimiento de tareas diarias con operaciones CRUD completas.
* **Stack Tecnológico:** Java, Spring Boot (REST API), Base de Datos Relacional (1 Tabla), HTML/CSS/JavaScript (Frontend).

---

### 📊 Proyecto 2 — Dashboard de Análisis
Tablero interactivo para la visualización de métricas de negocio, consolidación de datos y filtrado dinámico.
* **Stack Tecnológico:** SQL (Consultas avanzadas y agregaciones), Chart.js, HTML/CSS/JavaScript, Backend con conexión a Base de Datos (2-3 tablas relacionadas).

---

### 🔐 Proyecto 3 — Sistema de Autenticación
Módulo de seguridad empresarial enfocado en el control de acceso seguro y gestión de identidades.
* **Stack Tecnológico:** Java, Spring Security, JWT (JSON Web Tokens) o Sesiones, Base de Datos (Usuarios, Roles y Permisos).

---

### 📈 Proyecto 4 — Análisis de Ventas con Python
Proyecto de analítica de datos orientado a la carga, limpieza y exploración profunda de información comercial para la toma de decisiones.
* **Stack Tecnológico:** Python, Pandas, Matplotlib, Seaborn, Plotly, Jupyter Notebook.
